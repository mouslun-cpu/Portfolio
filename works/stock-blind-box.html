股票盲盒遊戲
FIG. 03 — 產品主畫面
關於這個作品

把「認識一家公司」變成抽盲盒的趣味體驗——隨機抽出一支股票，揭曉它的業務、定位與基本面，邊玩邊學。

用 React 打造，串接 Gemini API 生成公司介紹內容，是把學習包裝成遊戲的實驗作品。

作品亮點
把枯燥的選股學習變成抽盲盒的遊戲體驗
