關於這個作品

把零散的學術發表資料整理成一個互動式儀表板，能快速看出發表趨勢、領域分布與成果脈絡。

用 React 打造前端，串接 Gemini API 做資料整理與摘要，讓繁瑣的學術紀錄整理變成幾分鐘就能完成的事。
